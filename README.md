# Recipe Vision

## Описание проекта

**Recipe Vision** — это веб‑приложение, которое анализирует изображения продуктов (например, фотографию содержимого холодильника), автоматически определяет продукты и их количество, а затем предлагает подходящие рецепты с указанием калорийности.

Проект ориентирован на использование компьютерного зрения и backend‑логики для упрощения планирования питания и приготовления блюд.

---

## Основной функционал

* Загрузка изображения с продуктами
* Автоматическое распознавание продуктов на изображении
* Определение количества и единиц измерения продуктов
* Формирование списка доступных ингредиентов
* Генерация рецептов на основе распознанных продуктов
* Отображение калорийности каждого блюда

---

## Архитектура проекта

Проект построен по принципу разделения ответственности между фронтендом и бэкендом.

### Backend

* **FastAPI** — REST API
* **Pydantic** — схемы данных и валидация
* **Jinja2** — серверный рендеринг шаблонов (HTML)
* Обработка изображений и интеграция с ML‑моделью (логика выделена отдельно)

### Frontend

* HTML + Jinja2 шаблоны
* JavaScript для динамического отображения данных
* Отображение:

  * распознанных продуктов
  * списка рецептов

```json

  "recipes": [

      "ingredients": ["яйца", "молоко", "шпинат", "сыр"],
      "calories": 350,
      "advice": "Можно добавить специи по вкусу"
    }
  ]
}
```

## Установка и запуск

### Требования

* Python 3.10+
* pip / poetry

### Установка зависимостей

```bash
pip install -r requirements.txt
```

### Запуск сервера

```bash
uvicorn src.main:app --reload
```

После запуска приложение будет доступно по адресу:

```
http://127.0.0.1:8000
```
